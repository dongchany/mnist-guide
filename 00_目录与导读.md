# 从零开始的神经网络：手写数字识别实战教程

> **目标读者**：有高中数学基础（函数、导数、矩阵初步），了解基本 Python 语法的学习者。
>
> **最终成果**：你将亲手用纯 NumPy（不借助任何深度学习框架）实现两种神经网络，在真实的 MNIST 手写数字数据集上达到 90%+ 的识别准确率。
>
> **支持平台**：Windows、macOS、Linux。教程中涉及平台差异的命令均会分别标注。

## 如何使用本教程

**建议学习顺序**：按章节顺序阅读。每章末尾有练习题，建议动手完成后再进入下一章。

**代码约定**：所有代码使用 Python 3 + NumPy。代码注释为英文（与工业界习惯一致），讲解文字为中文。

**数学约定**：
- 向量用小写粗体：**x**
- 矩阵用大写粗体：**W**
- 标量用普通字体：y, n
- 求导用 ∂ 表示偏导数

**前置知识检查清单**：
- [ ] 知道什么是函数 f(x)
- [ ] 了解导数的基本概念（"变化率"）
- [ ] 听说过矩阵（不要求熟练运算，第 0 章会复习）
- [ ] 能写基本的 Python 代码（if/for/函数定义）
